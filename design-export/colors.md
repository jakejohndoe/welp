# Welp Color Palette

## Brand Colors (from index.css and Tailwind config)

### Primary Colors
- **Electric Blue (Primary)**: `#4169E1` / `hsl(231, 76%, 60%)` - Main blue for buttons, links, selected states
- **Golden Yellow (CTA)**: `#FFD700` / `hsl(51, 100%, 50%)` - Call-to-action buttons, business owner elements
- **Emerald Green (Success)**: `#10B981` / `hsl(158, 64%, 52%)` - Success states, positive feedback
- **Sky Blue (Secondary UI)**: `#42A5F5` / `hsl(207, 90%, 61%)` - Secondary UI elements
- **Teal (Accents)**: `#14B8A6` / `hsl(174, 72%, 56%)` - Accents and highlights

### Background & Text
- **White Background**: `#FFFFFF` / `hsl(0, 0%, 100%)` - Main background
- **Dark Text**: `#1A1A1A` / `hsl(0, 0%, 10%)` - Primary text color
- **Cream**: `hsl(36, 50%, 98%)` - Off-white background color

### UI Colors Used in Components
- **Blue-50**: Used in gradient backgrounds (`from-blue-50`)
- **Yellow-50**: Used in gradient backgrounds (`to-yellow-50`)
- **Gray-900**: `text-gray-900` - Headers and titles
- **Gray-600**: `text-gray-600` - Secondary text
- **Gray-500**: `text-gray-500` - Tertiary text
- **Gray-100**: `hover:bg-gray-100` - Hover states

### Specific Component Colors
- **Customer Blue**: `#4A90E2` - Customer account type cards
- **Business Yellow**: `#F5D033` - Business owner cards
- **Card Hover Border Blue**: `#4A90E2`
- **Card Hover Border Yellow**: `#F5D033`

### Background Colors with Opacity
- **Blue Background (10% opacity)**: `#4A90E2/10` - Light blue backgrounds
- **Yellow Background (10% opacity)**: `#F5D033/10` - Light yellow backgrounds

### Legacy Colors (for backward compatibility)
- **Mustard**: `hsl(45, 100%, 51%)`
- **Mustard Light**: `hsl(45, 100%, 71%)`
- **Mustard Dark**: `hsl(45, 100%, 41%)`
- **Buttercream**: `hsl(45, 61%, 85%)`
- **Burnt Orange**: `hsl(30, 70%, 47%)`
- **Charcoal**: `hsl(0, 0%, 17%)`
- **Brown**: `hsl(30, 20%, 35%)`

### System Colors
- **Border**: `hsl(45, 20%, 85%)`
- **Input**: `hsl(45, 20%, 92%)`
- **Ring**: `hsl(43, 87%, 45%)`
- **Destructive**: `hsl(0, 72%, 51%)`
- **Muted**: `hsl(45, 20%, 90%)`
- **Muted Foreground**: `hsl(0, 0%, 40%)`

### Gradients
- **Primary Gradient**: `linear-gradient(135deg, mustard, buttercream)`
- **Warm Gradient**: `linear-gradient(180deg, mustard, burnt-orange)`
- **Soft Gradient**: `linear-gradient(90deg, buttercream, cream)`
- **Accent Gradient**: `linear-gradient(135deg, sky-blue, sage-green)`
- **Background Gradient**: `from-blue-50 to-yellow-50` (Tailwind gradient)

### Shadows
- **Soft Shadow**: `0 4px 20px -2px mustard/10%`
- **Hover Shadow**: `0 10px 30px -5px mustard/20%`
- **Card Shadow**: `0 8px 24px -4px charcoal/8%`
